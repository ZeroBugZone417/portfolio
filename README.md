# Dineth Sudarshana - Personal Portfolio

A modern, responsive, and dark-themed personal portfolio website designed for a professional Web Developer. The website features a sleek glassmorphism aesthetic, smooth scroll-reveal animations, and a fully mobile-responsive layout.

## ✨ Features

- **Modern Dark Theme UI**: High-contrast, professional, and visually appealing dark mode structure.
- **Glassmorphism Design**: Beautiful translucent cards and elements using `backdrop-filter`.
- **Responsive Layout**: Adapts seamlessly to Desktop, Tablet, and Mobile screens using Flexbox and CSS Grid grids.
- **IntersectionObserver Animations**: Dynamic scroll-reveal effects that trigger as you scroll down the page.
- **Interactive Mobile Menu**: A smooth, full-screen mobile navigation toggle.
- **Dynamic Projects Grid**: Utilizes CSS `auto-fill` to perfectly align project cards regardless of grid count.
- **Contact Form Mockup**: Includes visual feedback and interactive UI for form submission.

## 💻 Technologies Used

- **HTML5**: Semantic tags and logical document structure.
- **CSS3**: Vanilla CSS utilizing custom variables, animations, gradients, and media queries.
- **JavaScript**: Vanilla JS for DOM manipulation, scroll events, and form handling mimicking a real backend interaction.
- **FontAwesome**: For lightweight and scalable SVG icons.
- **Google Fonts**: Utilizing 'Inter' for body text and 'Outfit' for headings to maintain modern typography.

## 🚀 How to Run Locally

Since this is a fully static website, no complex backend or build tools are required!

1. **Clone or Download** the repository to your local machine.
2. Navigate to the project folder (`portfolio`).
3. Simply double-click on `index.html` to open it in your default web browser.

*(Optional)* If you use VS Code, you can install the **Live Server** extension, right-click on `index.html`, and select *Open with Live Server* for real-time updates if you decide to edit the code.
